Игровое поле - это склад с лабиринтами коридоров. Грузчик должен расставить ящики на указанные места Цель - выполнить работу за наименьшее количество шагов. Грузчик может только толкать ящики по одному. Грузчик не может тянуть ящики и не может сдвинуть сразу два ящика.

Управление движениями грузчика осущеставляется стрелками/клавишами "W", "A", "S", "D" / кликом или касание по соседнему с грузчиком полю. Отмена последнего шага - клавиша "Escape" / двойной щелчок мышью или двойное касание.
